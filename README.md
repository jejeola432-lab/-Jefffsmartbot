# @Jefffsmartbot 🤖

A smart Telegram bot deployed on Railway with GitHub integration.

## Features
- ✅ Welcome message with inline keyboard
- ✅ Multiple commands (/start, /help, /ping, /info, /time, /echo)
- ✅ Smart responses to common messages
- ✅ Error handling
- ✅ 24/7 uptime on Railway
- ✅ Compatible with Python 3.11

## Deployment

### Prerequisites
- GitHub account
- Railway account
- Telegram bot token from @BotFather

### Steps
1. Fork this repository
2. Create a new project on Railway
3. Connect your GitHub repository
4. Add BOT_TOKEN as an environment variable
5. Deploy automatically!

## Commands
- `/start` - Welcome message
- `/help` - Help menu
- `/ping` - Check bot status
- `/info` - Bot information
- `/time` - Current server time
- `/echo <text>` - Echo your message
